Steps to install /deploy FiapProject:

1. Download zip package. This package will contain all the sources.
2. Download XAMPP suite from https://www.apachefriends.org/ro/download.html
3. Install XAMPP.
4. Unzip the package. Copy and paste the sources to htdocs folder from XAMPP instalation. (C:\xampp\htdocs\)
5. Start XAMPP Control Panel and start Apache/MySql services.
6. Open the browser and access project using the following link: http://localhost/FiapProject/
